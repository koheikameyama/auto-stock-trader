# ベンチマーク比較機能 デザイン

## 概要

ポートフォリオのパフォーマンスを日経225（ベンチマーク）と比較する機能。ダッシュボードの既存 NikkeiSummary コンポーネントを拡張し、定量指標の追加・AI分析への組み込みを行う。

## 要件

- **表示場所**: ダッシュボード（既存の NikkeiSummary コンポーネント）
- **ベンチマーク**: 日経225のみ（将来の拡張は容易）
- **チャート**: 既存のパフォーマンス比較チャート（日経 vs ポートフォリオ）を活用
- **定量指標**: 超過リターン・ベータ値・シャープレシオ（解説付き）
- **AI連携**: Daily Market Navigator で自然に言及

## 1. データ保存

### PortfolioSnapshot カラム追加

| カラム | 型 | 説明 |
|--------|-----|------|
| `nikkeiClose` | Decimal? | 日経225終値 |

- 日次スナップショット保存バッチで日経225終値を同時保存
- 既存スナップショットのバックフィルは yfinance から取得
- リターン率はAPI側で計算するため保存不要

### データフロー

```
日次バッチ（スナップショット保存）
  ├→ ポートフォリオ情報を計算・保存（既存）
  └→ lib/market-index.ts から日経225終値を取得して nikkeiClose に保存（新規）
```

## 2. 定量指標（NikkeiSummary 拡張）

### 表示する指標

| 指標 | 計算方法 | 初心者向け解説 |
|------|---------|-------------|
| 超過リターン | ポートフォリオリターン - 日経225リターン | 「日経平均より何%多く（少なく）稼げたか」 |
| ベータ値 | Cov(ポートフォリオ, 日経225) / Var(日経225) | 「市場全体が1%動いた時、あなたのポートフォリオが何%動くか。1より大きいと市場より激しく動く」 |
| シャープレシオ | (ポートフォリオリターン - 無リスク金利) / ポートフォリオの標準偏差 | 「リスクに対するリターンの効率。1以上で優秀」 |

### 評価基準

**超過リターン:**
- +3%以上: 良好（緑）
- -3%～+3%: 普通（グレー）
- -3%以下: 注意（赤）

**ベータ値:**
- 0.5未満: 安定型
- 0.5～1.0: バランス型
- 1.0～1.5: 積極型
- 1.5以上: ハイリスク

**シャープレシオ:**
- 1.0以上: 優秀（緑）
- 0.5～1.0: 普通（グレー）
- 0.5未満: 改善の余地あり（赤）
- ※無リスク金利は日本10年国債利回り近似で0.5%/年を使用

### UI

チャート下部にコンパクトカード形式で3指標を横並び表示。各カードタップでアコーディオン展開の解説を表示。

### 計算タイミング

- フロントエンド or 新規API `/api/portfolio/benchmark-metrics`
- PortfolioSnapshot の nikkeiClose + totalValue から日次リターンを計算
- 最低30日分のデータが必要（それ未満の場合は「データ不足」表示）

## 3. AI分析（Daily Market Navigator 拡張）

### プロンプトに追加するデータ

```
## ベンチマーク比較（直近1ヶ月）
- 日経225リターン: +2.3%
- ポートフォリオリターン: +5.5%
- 超過リターン: +3.2%
- ベータ値: 0.85
```

### AIへの指示

- `portfolioSummary` や `actionPlan` で、日経平均との比較に基づいたコメントを自然に含める
- 超過リターンがプラスの場合: ポートフォリオの強みを具体的に言及
- 超過リターンがマイナスの場合: 改善のヒントを提示
- 差が小さい場合は他の重要情報に文字数を割り当ててよい

### スキーマ変更

不要。既存の出力フィールド内で自然に言及。
