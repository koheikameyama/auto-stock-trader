# 自動売買ルール概要

## 戦略1: Breakout（出来高サージ + 高値ブレイク）

### エントリー条件
- 出来高サージ: 当日出来高 / 25日平均出来高 ≥ 2.0倍
- 高値ブレイク: 当日終値 > 過去20日高値
- 高値追い制限: 終値とHigh20の乖離 ≤ ATR × 0.5
- 翌日の確認足（confirmationEntry）でエントリー

### マーケットフィルター
- breadthフィルター: 銘柄のSMA25上回り比率 ≥ 73%（強い上昇相場のみ）

### 出口ルール
- 損切り: ATR × 1.0（最大3%キャップ）
- 建値撤退: ATR × 1.5 上昇でSLをエントリー価格に引き上げ
- トレーリングストップ: ATR × 2.5 上昇で発動、トレール幅 ATR × 1.5
- タイムストップ: 7営業日（トレーリング発動中は最大10日まで延長）
- 最大同時保有: 5ポジション

## 戦略2: Gap-Up（窓開け上昇 + 出来高）

### エントリー条件
- ギャップ: 始値が前日終値から3%以上上昇
- 陽線引け: 終値 ≥ 始値
- ギャップ維持: 終値 > 前日終値 × 1.03
- 出来高サージ: 当日出来高 / 25日平均出来高 ≥ 1.5倍
- 当日終値でエントリー（短期決戦）

### マーケットフィルター
- breadthフィルター: 銘柄のSMA25上回り比率 ≥ 60%（breakoutより緩い）
- 指数トレンド: N225 > SMA50

### 出口ルール
- 損切り: ATR × 1.0（最大3%キャップ）
- 建値撤退: ATR × 0.5 上昇でSLをエントリー価格に引き上げ
- トレーリングストップ: ATR × 1.0 上昇で発動、トレール幅 ATR × 0.5
- タイムストップ: 3営業日（トレーリング発動中は最大5日まで延長）
- 最大同時保有: 2ポジション（breakoutとは独立カウント）
- 平均保有日数: 約1.2日

## 共通ルール

### ユニバースフィルター
- 株価 ≤ 5,000円
- 25日平均出来高 ≥ 100,000株
- ATR% ≥ 1.5%
- クールダウン: 同一銘柄は直近エントリーから3日以上空ける

### ポジションサイジング
- 1トレードあたりリスク = 資金の2%
- 100株単位（損切り幅ベースで株数を逆算）

### VIXレジーム制限
- VIX < 20: 制限なし
- VIX 20-25: 最大2ポジション
- VIX 25-30: 最大1ポジション、デイトレモード
- VIX > 30: 新規エントリー停止、既存スイングは強制決済

### その他の取引停止条件
- CME先物乖離率 ≤ -3%: 全取引停止
- 日経平均前日比 ≤ -3%: 全取引停止
- 週次ドローダウン -5% / 月次 -10% / 5連敗: 取引停止
