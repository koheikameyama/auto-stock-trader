## リバランスルール

### **四半期チェック（3ヶ月ごと）**

1. **日本株システム**
   - 30万円未満：米国株から補填
   - 30-50万円：維持
   - 50万円超：超過分を米国株へ移動

2. **米国株**
   - 基本的に長期保有
   - 大幅な上昇時（+30%超）は一部利確検討

3. **現金**
   - 緊急バッファー：常時5-10万円
   - それ以上は投資に回す

---

## 税金の考慮

### **日本株システム**
- 譲渡益税：20.315%
- 年間利益50万円 → 税金10万円

### **米国株**
- 外国税額控除を活用
- 配当：米国10% + 日本20.315%（二重課税調整）
- 譲渡益：日本20.315%

## 結論

**3年目以降、資金が100万円を超えたらハイブリッド運用を推奨**

- 日本株システム：30-50万円（高年利103%を維持）
- 米国株：30-100万円（安定成長7-10%）
- 現金：緊急バッファー

**これにより：**
- 資金効率向上
- リスク分散
- 長期資産形成

**最初の2年間は日本株システムのみで実証してから移行することを推奨。**

## 関連分析スクリプト

この運用戦略の裏付けとなる数学的分析を行うスクリプトが `scripts/` に用意されています。

### **1. analyze-peak-positions.ts** - 同時保有数分析

**目的**: バックテスト結果から最大同時保有銘柄数を分析し、必要資金を見積もる

**使い方**:
```bash
npx tsx scripts/analyze-peak-positions.ts
```

**出力内容**:
- ピーク同時保有数（例：5枠）
- 平均同時保有数（例：1.8枠）
- 同時保有数の分布ヒストグラム
- ピーク時の資金配分詳細

**活用例**:
- 「初期資金50万円で足りるか？」の検証
- リスク管理（分散しすぎていないか確認）

---

### **2. compare-compound-vs-fixed.ts** - 複利 vs 固定資金の比較

**目的**: 複利運用（利益再投資）と固定資金運用（50万固定で利益出金）を10年間シミュレーション

**背景**:
- バックテストで「50万円=年利103%、75万円=年利30%」が判明
- 資金が増えると年利が激減する特性を持つ
- 「複利は常に有利」という常識が当てはまらない可能性

**使い方**:
```bash
npx tsx scripts/compare-compound-vs-fixed.ts
```

**出力内容**:
- 年次推移表（固定戦略 vs 複利戦略）
- 出金累計額
- 10年後の資産総額比較

**重要な発見**:
このシステムでは「固定50万円で高年利を維持し、利益を出金する戦略」が数学的に優位。

---

### **3. find-breakeven-point.ts** - 損益分岐点の精密計算

**目的**: 「何年目で複利が固定資金を逆転するか」を実測データ（7パターン）で計算

**実測データ**:
- 50万円 → 年利103.1%
- 75万円 → 年利90.3%
- 100万円 → 年利81.7%
- 150万円 → 年利80.3%
- （500万円まで計測）

**使い方**:
```bash
npx tsx scripts/find-breakeven-point.ts
```

**出力内容**:
- 15年間の年次推移
- 損益分岐点の年数と資金規模
- 資金規模別の年利・年間利益比較表

**戦略への示唆**:
- 短期（1-7年）：固定50万円が有利
- 中期（8-10年）：複利が逆転し始める
- 長期（11年以降）：複利が圧倒的に有利
- **推奨**: 3-5年は固定戦略で実績を作り、その後判断

---

## これらの分析が示す結論

1. **初期資金**: 50万円あれば十分（ピーク5枠保有でも対応可能）
2. **運用方針**: 最初の3-5年は固定50万円で年利103%を維持
3. **利益の使い道**: 出金して米国株インデックスへ分散投資
4. **長期戦略**: 日本株は50万円上限、それ以上は米国株で運用

この数学的根拠に基づき、上記のハイブリッド運用戦略を提案しています。

## システム拡張ロードマップ

### 拡張の原則

**拡張の前にライブ実績の証明が必須。** バックテストの数字がライブで再現できなければ、拡張しても意味がない。

### Phase 0：ライブ実績の証明（2026/04〜10）

現在のシステム（Gapup + PSC + Weekly-Break）のライブ性能を検証する。

| 検証項目 | 合格基準 | 見方 |
|---------|---------|------|
| スリッページ | BT想定比 ±1%以内 | シグナル価格 vs 実約定価格 |
| 約定率 | 90%以上 | シグナル発生数 vs 実約定数 |
| 実PF | 2.0以上 | BT 4.38からの劣化度 |
| 実年利 | 30%以上 | BT 103%からの劣化度 |
| 実最大DD | 15%以下 | BT 6.7%からの悪化度 |

**Phase 0クリア条件：年利30%以上 かつ PF 2.0以上**

---

### Phase 1：相関の低い戦略の追加（2026/10〜2027）

同じ日本株市場で、既存のモメンタム系と相関の低い戦略を追加する。

**なぜ「別の市場」より先か：**
相関の低い戦略を組み合わせると、リターンは足し算だがDDは足し算にならない。
これがCalmar Ratioを最も効率的に上げる方法。

#### 候補戦略

| 戦略タイプ | 概要 | 現戦略との相関 | 難易度 |
|-----------|------|--------------|--------|
| **平均回帰系** | RSI oversold + 出来高急増 → 逆張りエントリー | 低い | 中 |
| **セクターローテーション** | 業種別ETFの相対強度で乗り換え | 低い | 中 |
| **イベントドリブン** | 決算発表前後のボラティリティを狙う | 中程度 | 高 |

#### 実装の進め方

```
① 平均回帰系のバックテスト（2週間）
② 既存戦略との相関分析（相関係数 0.3以下が目標）
③ 合算ポートフォリオのCalmar Ratio検証
④ ライブ投入（既存と並行運用）
```

---

### Phase 2：米国株への展開（2027〜2028）

日本株で実証した戦略ロジックを米国市場に適用する。

#### なぜ米国株か

| 要素 | 日本株 | 米国株 |
|------|-------|--------|
| 流動性 | 中 | 最大（スリッページが小さい） |
| データの質・量 | 中 | 豊富（無料APIも多い） |
| 取引時間 | 9:00-15:30 | 23:30-6:00（日本時間）→ 夜間運用可能 |
| 銘柄数 | 約3,800 | 約8,000（選択肢が多い） |
| 空売り | 制約あり | 容易 |

#### 必要な準備

| 項目 | 具体的な対応 |
|------|------------|
| 証券口座 | Interactive Brokers（IB）開設 |
| データAPI | Polygon.io or Alpha Vantage |
| 戦略適用 | 既存ロジック（Gapup等）を米国株データでBT |
| 為替リスク | 円建て管理 or ドル建てで運用 |
| 税務 | 外国税額控除の確定申告が必要 |

#### 段階的な展開

```
① IB口座開設 + 米国株データ取得基盤構築（1ヶ月）
② 既存戦略の米国株バックテスト（1ヶ月）
③ 米国株固有の戦略開発（オーバーナイトギャップは米国でも有効な傾向）
④ 少額（$3,000〜）でライブ検証
⑤ 実績確認後、資金を増額
```

---

### Phase 3：インフラの高度化（2028〜・資金規模に応じて）

| 項目 | 内容 | 導入タイミング |
|------|------|--------------|
| ポートフォリオリスク管理 | 戦略横断のDD制御、相関モニタリング | 戦略3つ以上 |
| リアルタイム監視ダッシュボード | PnL・ポジション・リスク指標の可視化 | 運用資金300万超 |
| マルチブローカー対応 | 約定分散、障害時の冗長性 | 運用資金1,000万超 |
| アラート・異常検知 | 戦略劣化の自動検知、DD閾値アラート | 常時 |

### 拡張ロードマップ全体像

```
2026/04       Phase 0 開始（ライブ検証）
   │            └ 50万円・3戦略で実績計測
   │
2026/10       Phase 0 判定
   │            ├─ 合格 → Phase 1 開始
   │            └─ 不合格 → 戦略改善してから再検証
   │
2026/10〜2027  Phase 1（戦略追加）
   │            └ 平均回帰系を追加、相関分析
   │
2027〜2028    Phase 2（米国株展開）
   │            └ IB口座、既存戦略を米国でBT・ライブ投入
   │
2028〜        Phase 3（インフラ高度化）
                └ 資金規模に応じてリスク管理・監視を強化
```

**原則：各Phaseのクリア条件を満たしてから次へ進む。焦って拡張しない。**
